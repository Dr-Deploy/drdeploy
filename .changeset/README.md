# Changesets

Hi! This folder is automatically generated by [`@changesets/cli`](https://github.com/changesets/changesets). It contains:

- A config file (`config.json`)
- A bunch of `.md` files (one per pending change)

To add a new changeset for a change you want to ship, run:

```bash
bun run changeset
```

The interactive prompt will ask which packages changed, what bump type, and ask for a summary. The result lands as a `.md` file in this folder.

When `main` is tagged for release, CI runs `bun run release` which consumes these `.md` files, bumps versions, writes changelogs, and publishes to npm.

See [the Changesets docs](https://github.com/changesets/changesets/blob/main/docs/intro-to-using-changesets.md).
